## Behavioral notes worth knowing next session

1. **Deprecation predicate.** `formatReplDeprecationMessage` at `src/cli/replDeprecation.ts` is a pure helper taking `{ source, env }`. The call site in `src/main.ts` checks `resolution.surface === 'repl'` (not `effectiveSurface`) so the missing-binary fallback stays silent.
2. **Suppression mechanism.** Only `SOV_NO_DEPRECATION_WARNING=1` suppresses. Strict-equal — `'0'`, `''`, `'true'` all leave the warning enabled (documented behavior; tested by `replDeprecation.test.ts`).
3. **M13 is the next milestone** — removes `src/ui/terminalRepl.ts` + the readline input helpers it depends on. Per Postmortem Rule 4, M13 ships after M12's deprecation has soaked in real use.
4. **No new wire types or protocol changes** in M12. The TUI side is unaware that anything happened — main.ts emits a stderr line before launching either surface; the post-emit behavior is identical to M11.5.
