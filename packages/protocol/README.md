# @yevgetman/sov-protocol

Typed wire protocol for the Sovereign AI gateway (Contract #2) — the SSE event
union, the endpoint request/response shapes, and a thin, dependency-free
`fetch`-based client. A single source of truth for any consumer (the Go TUI,
`sov drive`, external apps) that talks to a running `sov gateway`.

Runs on Node ≥20 and Bun ≥1.2. Zero runtime dependencies.
